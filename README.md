# Notion Auto Manager

Automação desenvolvida em Node.js para gerenciamento automático de atividades do Notion.

## Funcionalidades

- Preenche automaticamente a Data Validação
- Conclui atividades após 15 dias
- Executa automaticamente via GitHub Actions

## Tecnologias

- Node.js
- GitHub Actions
- Notion API

## Variáveis

NOTION_TOKEN

NOTION_DATASOURCE_ID
